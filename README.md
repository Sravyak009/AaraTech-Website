# AaraTech-Website
Project Description:-
1.This project is a professional IT company website developed for AaraTech Digital Solutions Pvt. Ltd. as part of a Web Development Internship.
2.The website provides information about the company, its services, industries served, portfolio, careers, blog, and contact details. It helps customers learn about the company and contact the business through the Contact Us page.

Technologies Used:-
- HTML5
- CSS3
- JavaScript
- Node.js
- Express.js
- MongoDB
- Git & GitHub

Website Pages:-
1.Home -
Displays the company introduction, hero section, services overview, testimonials, and call-to-action.
2.About -
Provides company information, mission, vision, and objectives.
3.Services -
Displays all IT services offered by the company.
4.Industries -
Shows the industries served by the company.
5.Portfolio -
Displays completed projects and company achievements.
6.Careers -
Shows job openings and career opportunities.
7.Blog -
Displays technology-related articles and updates.
8.Contact -
Allows customers to contact the company through a contact form.

Features :-
- Responsive Design
- Professional User Interface
- Multiple Web Pages
- Easy Navigation
- Contact Form
- Backend Integration
- Database Connectivity
- Mobile Friendly
